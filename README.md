# Website - Advogado do Consumidor

Acesse o site [aqui](https://advleo.com.br/).

## Desenvolvimento

- Desenvolvido em **React.jsx** para uma experiência dinâmica e responsiva.
- Implementação da biblioteca **Helmet** para otimização de SEO.
- Hospedagem realizada na **Hostinger**, garantindo desempenho estável.
- Estudo detalhado das melhores **Google Tags** para monitoramento eficaz.
- Integrado ao **Google Search Console** para análise de performance e indexação.
- Suporte para carregadores de módulos **AMD/CMD** (Require.js & Sea.js), com plugins personalizados para edição e definição de conteúdo.

## Design

- **Paleta de Cores**:
  - <span style="color:#f1f952;">`#f1f952`</span> (Amarelo vibrante) para elementos de destaque, trazendo visibilidade e contraste.
  - <span style="color:#000000;">`#000000`</span> (Preto) para o fundo principal, criando um visual moderno e sofisticado.
  - <span style="color:#ffffff; background-color:#000000;">`#ffffff`</span> (Branco) para o texto principal, garantindo legibilidade.
  - <span style="color:#d3d3d3;">`#d3d3d3`</span> (Cinza claro) para áreas secundárias e divisões sutis.
  - <span style="color:#1d1e20;">`#1d1e20`</span> (Cinza escuro) para criar equilíbrio com a tonalidade mais forte de fundo.

- **Fontes**:
  - **Roboto**, uma tipografia moderna e limpa, utilizada para proporcionar uma leitura confortável e acessível.
